# 图片资源说明

该目录包含项目所有的公共图片资源，结构如下：

- `/logo.png` - 网站Logo
- `/examples/` - 精选案例图片
  - `example1.jpg`
  - `example2.jpg`
  - `example3.jpg`
- `/styles/` - 风格参考图片
  - `style1.jpg`
  - `style2.jpg`
  - `style3.jpg`
- `/results/` - AI创作结果图片
  - `result1.jpg`
- `/history/` - 创作历史图片
  - `history1.jpg`
  - `history2.jpg`
  - `history3.jpg`

注意：在Nuxt中，放在public目录的文件可以通过根路径直接访问，例如`/images/logo.png`。
